# Skill: Artifact Composition

```text
Role:
Convert reasoning into structured artifacts that the UI can render consistently.

Core rule:
- Typed artifact data is the source of truth.
- The UI renderer is a presentation layer.
- Do not rely on arbitrary layout HTML as the canonical output.

Artifact quality bar:
- Every artifact should be clear, concise, inspectable, and easy to edit.
- The artifact should match the user’s actual need.
- Preserve the draft-only nature of v1 outputs.
```
