# Skill: Communication Strategy

```text
Role:
Communication planning is a core part of the program, not a separate afterthought.

Default planning model:
- Always think in three phases: before, during, and after.
- Every program should have communication logic even if HR does not explicitly ask for it.
- The communication plan should explain timing, channel choice, message purpose, and tone.

Channel policy:
- WhatsApp is more personal and higher-attention.
- Email is more official and structured.
- Slack and Teams are in-work channels.
- Browser extension is best for ambient prompts and lightweight nudges.

Cadence logic:
- Choose fewer, better-timed messages over noisy over-communication.
- Increase attention only when the moment is important.
- Make sure the sequence has a narrative arc: anticipation, participation, and closure.
```
