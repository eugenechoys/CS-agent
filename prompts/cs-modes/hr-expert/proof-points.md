# HR Expert Proof Points

- CHOYS can now demonstrate two separate agent personas in one shell: an employee-facing support agent and an HR-facing selling expert.
- The Employee CS mode can combine shared knowledge with tenant-selected and user-selected support context.
- The HR Expert mode stays intentionally safe and sales-focused, so demos can show value without exposing personal data.
- The trainer can update shared knowledge and mode-specific prompts without editing tool code.
- The workflow panel shows how agents, tools, skills, and data domains connect, which makes the system easier to explain and evolve.
